Principais Formatações

Negrito / Destaque

Para escrever um termo em negrito, como esse, utilize a tag B (não semântica).

Para escrever um termo em destaque, como esse, utilize a tag STRONG (semântica).

Itálico / Ênfase

Para escrever um texto em itálico, como esse, utilize a tag I (não semântica).

Para escrever um texto em ênfase, como esse, utilize a tag EM (semântica).

Texto Marcado

Ou para escrever um texto e marcá-lo, tipo assim, utilize a tag MARK.

Texto Deletado

Para escrever que um texto foi excluído e indicar que ele deve ser lido, mas não considerado, utilize a tag DEL

Texto Inserido

Para escrever um texto que foi inserido e indicar que ele foi adicionado depois, utilize a tag INS

Também existe a tag U (não semântica).
